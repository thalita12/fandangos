Introdução ao LESS

LESS é um pré-processador de CSS que estende mais funcionalidades ao CSS, como variáveis e mixins.

Você escreve um arquivo de extensão ‘.less’, mas usará no final um de ‘.css’.

Eles servem para facilitar a escrita de CSS, tornando-o mais inteligente, flexível e de fácil manutenção.

Instalação

Pré-requisito: é necessário ter o Node.js na sua máquina.

Comando para instalar o LESS:

$ npm install less -g

Como usar

Mãos à obra!

Para compilar os arquivos LESS, execute o seguinte comando:

$ grunt style

Para que os arquivos sejam compilados automaticamente, execute o comando abaixo:

$ grunt watch
